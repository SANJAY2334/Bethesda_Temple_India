# User Guide - Grace Harbor Church Website

Welcome! This guide helps non-technical users access and use the website.

Accessing the site
- Open: https://your-deployed-site.example or for local testing: http://localhost:5173/

Register & Login
- Registration: Public registration may be disabled. Contact admin if account needed.
- Login: Click Admin → enter email and password. Password must be 12+ chars.

Using features
- View Sermons: Navigate to Home → Sermons section. Click a sermon to view details.
- Watch Livestream: If a livestream is active, a player appears on the homepage.
- Donate: Go to Donate page → enter amount and email → follow payment flow.
- Submit Prayer Request: Visit Prayer page → fill form. Toggle confidential if private.

Expected behavior
- After successful donation, you will receive a confirmation email.
- Confidential prayer requests are only visible to admin and assigned prayer team members.

Common errors & solutions
- Payment failed: Check card details and try again; contact support with payment ID.
- Login failed: Reset password with admin or ensure correct credentials.
- Page not loading: Verify internet connection and try again later.

Support
